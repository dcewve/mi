# 中粮生化（成都）年会抽奖活动页面

一个纯前端的年会抽奖页面，支持：

- 四个奖项等级：特等奖、一、二、三等奖
- 自动 / 手动两种抽奖模式（自动模式会在设定时间后自动停止）
- 每一轮可设置抽取人数，所有人员概率相同
- 已中奖人员自动从池中移除，不再参与后续抽奖
- 抽奖开始前不显示姓名，进行时姓名闪烁轮播，定格后显示中奖人
- 可随时添加 / 移除参与人员

## 使用方式

1. 在本项目目录下启动一个静态服务器，例如：

```bash
cd lottery-annual-party
npm install -g serve # 如果还没安装
serve .
```

或使用项目内脚本（需要本地 Node 环境）：

```bash
cd lottery-annual-party
npm install
npm run start
```

2. 在浏览器中访问命令行提示的地址（例如 `http://localhost:3000`）。

3. 按页面上的提示添加抽奖人员，设置奖项和模式，即可开始抽奖。
